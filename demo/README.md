# WordPress UI Theme - Kitchen Sink Demo

This demo showcases all available widgets, templates, and features of the WordPress UI theme for react-jsonschema-form.

## Running the Demo

1. Install dependencies:
```bash
cd demo
npm install --legacy-peer-deps
```

2. Start the development server:
```bash
npm run dev
```

The demo will open in your browser (typically at `http://localhost:3000`, or the next available port if 3000 is in use)

## What's Included

The kitchen sink demo includes examples of:

### Widgets
- Text inputs (text, password, email, URL)
- Number inputs (number, updown, range)
- Textarea
- Select dropdown
- Radio buttons
- Checkbox (single)
- Checkboxes (multiple)
- Date/Time inputs (date, datetime, time)
- Hidden field

### Templates
- Field templates with labels and help text
- Object fields (nested objects)
- Array fields (with add/remove functionality)
- Error handling and validation
- Required field indicators

### Features
- Form validation
- Real-time form data display
- Submitted data preview
- All WordPress component styling

## Testing

Use this demo to:
- Verify all widgets render correctly
- Test form validation
- Check WordPress styling consistency
- Test form submission
- Verify all templates work properly
